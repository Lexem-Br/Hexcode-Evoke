### ✦ Description:

Hexcode Evoke is an expansion for the Hexcode mod by Riprod, where you can bring objects to life that will follow you on your adventure. 𖥔 ݁ ˖

### ✧ Notes:

*   The project is open source, so feel free to use the assets in other projects. Credits are not required, but they are appreciated.
*   You don't need Hexcode to use the main features of Hexcode Evoke, but it's recommended for a complete experience.

## ✦ How to use:

Hex items can be crafted at the arcane and alchemy bench.

### ✧ Evoke Book

A simple way to bring Hex Creatures to life, just use Evoke Book on a hex object.

### ✧ Hexcode Gliph

Combine the glyphs projectile → area → evoke to create an area of effect. Evoke can be created by combining square + triangle + diamond. For more information about the Hexcode system, visit: [https://www.curseforge.com/hytale/mods/hexcode](https://www.curseforge.com/hytale/mods/hexcode)
